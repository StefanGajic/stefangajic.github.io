HTML and CSS

HTML (the Hypertext Markup Language) and CSS (Cascading Style Sheets) are two of the core technologies for building Web pages. HTML provides the structure of the page, CSS the layout, for a variety of devices.

Bootstrap and SASS

Bootstrap is the most popular CSS Framework for developing responsive and mobile-first websites. SASS (Syntactically Awesome Style Sheets) is an extension of CSS that enables us to use variables, nested rules and more. It also helps to keep things organised and allows you to create style sheets faster.

Go programming language

Go is a statically typed, compiled programming language designed at Google by Robert Griesemer, Rob Pike, and Ken Thompson. Go is syntactically similar to C, but with memory safety, garbage collection, structural typing, and CSP-style concurrency.

Ruby and Ruby on Rails

Ruby is an interpreted, high-level, general-purpose programming language. It was designed and developed in the mid-1990s by Yukihiro "Matz" Matsumoto in Japan. Ruby is dynamically typed and uses garbage collection. Ruby on Rails, or Rails, is a server-side web application framework written in Ruby under the MIT License. Rails is a model–view–controller (MVC) framework, providing default structures for a database, a web service, and web pages.
